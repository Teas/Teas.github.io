地块审批

发表于 2018-02-26

前置条件如下

1:地块需要绑定区域
2:需新增审批流程
3:地块初始化参数

新增审批流程步骤

1.创建流程类别。流程名称必须为地块审批
http://localhost:8080/charge_ChargeApprovalProcess_processCategoryList.do

2.新增部门
http://localhost:8080/charge_ChargeApprovalProcess_processDepartmentList.do

3.新增流程
http://localhost:8080/charge_ChargeApprovalProcess_processList.do

地块初始化参数

InvestProjectBean在创建的时候需要初始化以下参数 version=1;taskId=1;approvalStatus=1

相关接口

发起审批

http://127.0.0.1:8080/rest/investprojectInfo/startProjectApprovalProcess

参数 取值
taskId InvestProjectBean中的taskId
version 版本号。InvestProjectBean中的version
userId 操作人id

审批流程

流程发起后可进行审批,拒绝操作

http://127.0.0.1:8080/rest/investprojectInfo/runProjectApprovalProcess

参数 取值
taskId taskId
taskOperation 描述
taskState 审核状态 0-拒绝,1-通过,2-待处理
userId userId
version 版本

查询审批

http://127.0.0.1:8080/rest/investprojectInfo/getTaskProcessStepDetail?taskId=111

参数 取值
taskId taskId
